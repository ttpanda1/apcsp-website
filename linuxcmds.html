Linux Reference Page
Name Description Example of Usage
sudo Allows a permitted user to execute a command as a superuser or another user Used when a commands needs the root's permission
i.e.
sudo emacs index.html
ls Lists the contents of the current directory Used when you want to see contents of the directory
i.e.
ls
cd Changes the directory you are in Used whenever you wish to go into another directory
i.e.
cd home
mkdir Creates a new directory Used whenever you wish to create a new folder or subfolder
i.e.
mkdir test
cp Copies a file or directory Used whenever you wish to copy and paste a file or directory (good for oragnization)
i.e.
sudo cp index.html linuxcmds.html
rm Removes a file or directory Used whenever you wish to remove a file or directory
i.e.
rm index.html
cat Views the text/code of a file Used whenever you wish to see the text inside a file
i.e.
cat index.html
echo Displays a string on standard output Used whenever you wish to display a line of text
i.e.
echo hello world
uname Prints certain system information Used when you wish to see certain information about the system
i.e.
uname -a
mv Renames a source or moves the source(s) to a directory Used when you wish to rename a file/folder or move a file/folder into another directory
i.e.
sudo mv IMG_1234.JPG Testpicture.jpg
sudo mv IMG_1234.JPG /var/www/html/sp1a/images
